HTML 标记标签通常被称为 HTML 标签

HTML 标签是由尖括号包围的关键词

HTML 标签通常是成对出现

标签对中的第一个标签是开始标签，第二个标签是结束标签

开始和结束标签也被称为开放标签和闭合标签

标签是成对出现，比如<b>和</b>
